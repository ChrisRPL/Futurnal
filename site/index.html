What Futurnal is

A local-first engine for personal knowledge: ingestion pipelines, graph memory, embeddings, temporal reasoning, and an interface for search, chat, and insights.

Hybrid Search

Semantic, graph, and temporal retrieval in one query flow.

Grounded Chat

Responses constrained to your indexed corpus and graph context.

Connectors

Obsidian, IMAP, GitHub, and local files.

Privacy

Local-first by default with explicit consent boundaries.
